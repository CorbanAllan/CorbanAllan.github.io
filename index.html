Sequencio

Sequencio is a modern, visually engaging memory game app where players replicate increasingly complex sequences of colored quarters. The app features smooth animations, immersive sound effects, and a global leaderboard that lets players compete worldwide.

Key Features:
• Classic sequence memory gameplay that increases in difficulty each round
• Global leaderboard powered by Firebase Firestore
• Neon-style visuals with glowing effects and immersive audio
• Settings screen with toggles for sound, music, and vibration
• Google Sign-In authentication to securely sync scores and preferences
• Responsive UI that adapts smoothly across device sizes

Technologies Used:
Kotlin & Jetpack Compose · Firebase Authentication · Firebase Firestore · Hilt · Coroutines & StateFlow · Material 3 Components
